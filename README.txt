# Telegram Bot Creator

## Как запустить

1. Установи зависимости:
```
pip install -r requirements.txt
```

2. В `bot/main.py` укажи свои данные:
- `API_ID`
- `API_HASH`
- `TG_BOT_TOKEN`

3. Запусти бота:
```
python bot/main.py
```

## Как пользоваться

1. Отправь `.session` файл в Telegram-бота.
2. Получи токен созданного бота и номер аккаунта.

**Важно:** Telegram может ограничивать создание ботов при слишком частом использовании.
